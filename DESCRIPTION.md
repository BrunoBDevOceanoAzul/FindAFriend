# FindAFriend 🐾

**FindAFriend** é uma API REST desenvolvida em Node.js com TypeScript e Fastify para conectar organizações de adoção a pessoas interessadas em adotar pets.

## Funcionalidades

- Cadastro e autenticação de organizações (ORGs)
- Cadastro e gerenciamento de pets disponíveis para adoção
- Busca de pets por cidade, porte, nível de energia e ambiente ideal
- Visualização detalhada de cada pet

## Stack

- **Node.js** + **TypeScript** — Tipagem estática e código robusto
- **Fastify** — Framework web rápido e eficiente
- **Prisma** — ORM moderno com PostgreSQL
- **Zod** — Validação de schemas
- **Vitest** — Testes unitários e de integração
